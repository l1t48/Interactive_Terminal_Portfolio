### Terminal Portfolio with Animated Profile Card

## Project Description
A unique terminal-inspired personal portfolio showcasing projects, skills, and experience. Built with React + Vite, featuring a 3D animated profile card for a modern interactive touch. It combines a nostalgic command-line experience with modern web technologies, including a 3D animated profile card. The modular architecture ensures scalability and easy customization for new projects or sections.

## Features
- Terminal-style interface for an interactive developer experience
- 3D animated profile card
- Responsive design that adapts to different screen sizes
- Modular architecture for easy project additions

## Requirements
- Node.js (v20.19.0+)
- npm

## Running the Project Locally
- npm install
- npm run dev
